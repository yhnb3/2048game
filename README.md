### skills

- React

- Typescript

- Jest

  테스트 코드를 작성하며 개발을 해보았습니다.

- Scss

- Store

  로컬스토리지를 조금 더 편하게 사용하기 위해 사용

### 개발 후기

1. 테스트 코드를 짜며 작성을 했는데 생각보다 짜기도 힘들뿐 더러 작은 사이즈의 개발이라 그런지 필요성이 크게 느껴지지 않아서 조금 힘들었습니다.
2. 테스트 코드를 타일을 움직이는 것 위주로 짜다 보니 함수위주만 테스트 코드를 짰습니다.
3. 게임을 구현하면서 가장 중요하게 생각한 부분은 타일이 움직일 때의 가시성이었고 그 부분을 참조한 게임을 유사하게 따라하려고 노력하였습니다.
4. scss 믹스인을 이용하여 반응형으로 개발하였습니다.
